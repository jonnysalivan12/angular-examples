④ AGENT
TESTER
Inżynier QA · wyższa temperatura
🧪 Test Generator
Pisze testy happy path i negatywne. Wyższa temperatura → kreatywne edge cases.
🔴 Error Translator
Kondensuje stack trace → krótki komunikat. Ogromna oszczędność tokenów w pętli.
INPUT: kod od CODER + AC od CLARIFY
⟲ PĘTLA POPRAWEK:  TESTER (Error Translator) → kondensuje błąd → CODER (Self-Corrector) → poprawka → TESTER … aż do zaliczenia wszystkich testów
📉
Zarządzanie kontekstem
Context Manager → Coder dostaje tylko niezbędne pliki, nie całe repozytorium. Minimalizacja zużycia tokenów.
🧩
Separation of Concerns
Coder nie zna wymagań biznesowych — dostaje gotowy abstrakt techniczny. Każdy agent widzi tylko swój wycinek.
🌡️
Temperatura modelu
Coder: niska (przewidywalny, deterministyczny kod). Tester: wyższa (kreatywne, nieprzewidywalne edge cases).
🔀
DAG zależności
Sequence Router tworzy graf skierowany acykliczny — gwarantuje poprawną kolejność generowania kodu.
⚡
Oszczędność tokenów
Spec Writer + Error Translator eliminują szum informacyjny na wejściu i wyjściu najbardziej kosztownych agentów.
